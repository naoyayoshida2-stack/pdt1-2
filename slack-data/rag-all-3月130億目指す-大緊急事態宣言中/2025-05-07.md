↓のクエリを今日回したら2月中旬以降のPDT2エントリーCSが0になってしまったんだけど、これなんでか確認できる？

```--変数を宣言する
declare start_date date default date('2024-10-01');
declare end_date date default current_date() - 1;

with ro2_raw as (
  select
    case
      when entry_start_type = 'independent_entry_form' then '独立EF'
      when entry_start_type = 'reentry_form' then '独立EF'
    else entry_start_type end as entry_start_type,
    pdt2_user_flag,
    date_trunc(entry_start_date,week) as entry_week,
    sum(1) as sorting_uu,
    count(distinct jobseeker_id||jobseeker_branch_id) as entry_uu
  from
  dharma-dwh-rag.datamart.v_rag_entry_users_for_ro2
  where
  entry_start_date >= start_date
  group by all
)

select * from ro2_raw
order by 1,2,3```
